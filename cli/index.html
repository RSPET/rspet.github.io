CLI Commands

RSPET's server module provides a Command Line Interface (CLI) to its user. This part aims to introduce RSPET's CLI and detail the available commands.

The commands listed bellow are part of the essentials, the files and the udp plug-ins.

Feel free to ignore the "State(s)" and the "Transition" fields as the will rarely be of use to an end user and are here purely to assist contributors and Plug-in developers.

essentials

help

Desc : List commands available in current state or provide syntax for a command.

Syntax : help [command]

State(s) : "basic", "connected", "selected"

Transition : None

Choose_Host

Desc : Select a single host.

Syntax : Choose_Host

State(s) : "basic"

Transition : "connected"

Select

Desc : Select multiple hosts.

Syntax : Select

State(s) : "basic"

Transition : "multiple"

ALL

Desc : Select all hosts.

Syntax : ALL

State(s) : "basic"

Transition : "all"

Quit

Desc : Quit the CLI and terminate the server.

Syntax : Quit

State(s) : "basic"

Transition : None

Execute

Desc : Execute system command on client.

Syntax : Execute

State(s) : "connected", "multiple"

Transition : None

Close_Connection

Desc : Kick the selected Client(s).

Syntax : Close_Connection

State(s) : "connected", "multiple"

Transition : "basic"

List_Hosts

Desc : List all connected hosts.

Syntax : List_Hosts

State(s) : "basic"

Transition : None

List_Sel_Hosts

Desc : List selected hosts.

Syntax : List_Sel_Hosts

State(s) : "connected", "multiple"

Transition : None

Exit

Desc : Unselect all hosts.

Syntax : Exit

State(s) : "connected", "multiple"

Transition : "basic"

KILL

Desc : Stop client(s) from doing the current task.

Syntax : KILL

State(s) : "connected", "multiple"

Transition : None

files

Pull_File

Desc : Pull a regular text file from the client.

Syntax : Pull_File [local_file]

State(s) : "connected"

Transition : None

Pull_Binary

Desc : Pull a binary file from the client.

Syntax : Pull_Binary [local_bin]

State(s) : "connected"

Transition : None

Make_File

Desc : Send a regular text file to the host(s).

Syntax : Make_File [remote_file]

State(s) : "connected", "multiple"

Transition : None

Make_Binary

Desc : Send a binary file to the host(s).

Syntax : Make_Binary [remote_bin]

State(s) : "connected", "multiple"

Transition : None

udp

UDP_Flood

Desc : Flood target machine with UDP packets.

Syntax : UDP_Flood [payload]

State(s) : "connected", "multiple"

Transition : None

UDP_Spoof

Desc : Flood target machine with UDP packets via spoofed ip & port.

Syntax : UDP_Spoof [payload]

State(s) : "connected", "multiple"

Transition : None
