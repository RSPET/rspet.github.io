    About RSPET.

    RSPET (Reverse Shell and Post Exploitation Tool) is a Python based reverse shell with functionalities that assist in a post exploitation scenario. It's available to download (as source) through it's GitHub Repository as well as the repos of BlackArch and ArchStrike. RSPET is fully Open Source (released under the MIT license) and open to contributors.

    Key Features

    RSPET's key features include:

    • TLS encryption of Server-Client communication
    • Able to accept and manage multiple clients simultaneously
    • Full support for plug-ins on the Server module
    • An API for the Server module
    • A RESTful WebAPI for the Server module
    • An installation script for the Server module that will take care of all dependencies and key generation
    • Both Server and Client fully Open Source
    • Extensive Documentation on both the code/design and the usage/expansion/plug-in development
